### [FITCHA - 함께하는 운동 챌린지 플랫폼](https://github.com/yunyeook/Fitcha)
> SSAFY 13기 관통 프로젝트 (2인 팀 프로젝트)  
> 2025.04 ~ 2025.05 | Spring Boot + Vue.js 기반 웹 애플리케이션

**프로젝트 개요**  
혼자 하던 운동을 "함께하는 챌린지"로 바꾸어 지속성과 재미를 높여주는 소셜 기반 운동 플랫폼

**주요 기술 스택**  
`Java`, `Spring Boot`, `JPA`, `MySQL`, `Vue.js`, `STOMP`, `OAuth2`, `OpenAI API`, `YouTube API`

**핵심 기능**
- 챌린지 생성/참여, 실시간 진행률 시각화, 참여자 랭킹
- 운동 인증글 작성 및 댓글, 좋아요 기능
- STOMP 기반 실시간 채팅 기능
- 소셜 로그인 (Google, Kakao, Naver) 및 랜덤 닉네임 생성
- OpenAI API를 활용한 챌린지 소제목 자동 생성
